#### 9.常用的快捷键

##### 命令模式快捷键（按 Esc 键开启）:

|     名称      |             作用             |
| :-----------: | :--------------------------: |
|       A       |       在上方插入新单元       |
|       B       |       在下方插入新单元       |
| Shift + Enter |   运行本单元，选中下个单元   |
| Ctrl + Enter  |          运行本单元          |
|  Alt + Enter  | 运行本单元，在其下插入新单元 |
|       M       |    单元转入 markdown 状态    |

##### 编辑模式快捷键（ 按 Enter 键启动）:

|     名称      |         作用         |
| :-----------: | :------------------: |
| Ctrl+Shift+ - | 按光标所在行进行分割 |
